**层级**: 第一层 - 核心规则（补充）
**更新**: 2026-03-26 04:00 GMT+8

---

## 一、禁入规则

1. **不执行任何形式的下单指令**（包括模拟账户）
2. **不存储完整资金账户信息**（卡号、密码、验证码）
3. **不在无日志情况下执行高风险操作**（单次影响超过总仓10%的操作）
4. **不向未授权第三方透露用户持仓信息**

---

## 二、数据口径规则

### 涨跌幅口径
| 分类 | 阈值 | 说明 |
|------|------|------|
| 正常波动 | ±3% | 无需特别关注 |
| 异常波动 | ±5% | 需记录 |
| 剧烈波动 | ±9%以上 | 需写入当日报告 |
| 涨停 | +10%（主板）/ +20%（创业板/科创板）| 单独标注 |
| 跌停 | -10% / -20% | 单独标注 |

### 量比口径
| 分类 | 阈值 |
|------|------|
| 正常 | < 2.0 |
| 放量 | 2.0 ~ 5.0 |
| 异常放量 | > 5.0 |

### 换手率口径
| 分类 | 阈值 |
|------|------|
| 正常 | < 5% |
| 偏高 | 5% ~ 10% |
| 异常 | > 10% |

## 三、报告输出规范

### 文件命名
```
reports/daily-market-YYYY-MM-DD.md      # 市场开盘报告
reports/daily-close-YYYY-MM-DD.md        # 收盘报告
reports/weekly-YYYY-WXX.md               # 周报
reports/review-YYYY-MM-DD.md             # 复盘报告
```

### 报告头部格式
```markdown
# 市场日报 - YYYY年MM月DD日

**交易日**: ✅ 是 / ❌ 否
**市场概况**: 上涨/下跌/震荡
**上证**: +X.XX%
**深成**: +X.XX%
**创业板**: +X.XX%
**科创50**: +X.XX%
**两市成交额**: XXXX亿
**涨跌停**: 涨停 XX家 / 跌停 XX家
```

## 四、关注列表管理

### 建仓观察列表
用户持仓或关注的股票代码列表，存放在：
`data/watchlist.yaml`

### 异动标记规则
- 单日涨跌幅超过 ±5% → 标记「异动」
- 单日量比超过 5 → 标记「放量」
- 触及跌停/涨停 → 标记「异动-涨跌停」

---

## 五、风险阈值

| 场景 | 阈值 | 处理方式 |
|------|------|----------|
| 单日亏损提醒 | -3%（相对持仓成本）| 主动通知用户 |
| 单日暴涨提醒 | +7%（相对持仓成本）| 主动通知用户 |
| 持仓占比过高 | 单只 > 30% 总仓位 | 通知用户注意分散 |
| 连续下跌 | 连续3日下跌 | 生成持仓分析 |

---

## 六、不更新时间

以下情况不触发数据更新：
- 节假日（以中国证监会公告为准）
- 非交易时段（正常情况只更新9:30和16:00）
