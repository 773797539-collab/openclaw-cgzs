## 一、已建立的工作流

### 每日定时任务链
```
cron trigger (systemEvent)
  → main 接收并解析任务类型
  → stock-exec 执行数据获取
  → stock-review 验收结果
  → main 汇总并通知用户
```

### 手动任务链
```
用户消息 → main
  → stock-research（可选研究）
  → stock-exec（执行）
  → stock-review（验收）
  → main（通知用户）
```
