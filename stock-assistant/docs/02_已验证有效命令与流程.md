## 二、MCP HTTP 接口

### 地址
```
http://82.156.17.205/cnstock/mcp
```

### Python 调用封装（mcp_stock.py）
```python
from scripts.mcp_stock import get_stock_info, get_medium_info, call_mcp
```

### 直接 curl 调用
```bash
# brief（价格+PE/PB+名称）
curl -s -X POST 'http://82.156.17.205/cnstock/mcp' \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"brief","arguments":{"symbol":"SH605365"}},"id":1}'

# medium（+资金流向+换手率+多周期均量）

  -H 'Content-Type: application/json' \
```

## 三、东财 push2 API

### 地址
```
https://push2.eastmoney.com/api/qt/clist/get
https://push2.eastmoney.com/api/qt/stock/get
```

### ⚠️ 必须加 Referer 头（否则连接被重置）
```bash
curl -s --max-time 8 "https://push2.eastmoney.com/api/qt/stock/get?secid=1.000001&fields=f43,f57,f58,f170" \
  -H "User-Agent: Mozilla/5.0" \
  -H "Referer: https://quote.eastmoney.com/"
```

### 涨停股池查询
```python
import subprocess, re, json

URL = ("https://push2.eastmoney.com/api/qt/clist/get"
       "?pn=1&pz=100&po=1&np=1&fltt=2&invt=2&fid=f3"
       "&fs=m:0+t:6,m:0+t:13,m:1+t:2,m:1+t:23"
       "&fields=f12,f14,f3,f8,f10&cb=x")
cmd = ["curl","-s","--max-time","8", URL,
       "-H","User-Agent: Mozilla/5.0",
       "-H","Referer: https://quote.eastmoney.com/"]
r = subprocess.run(cmd, capture_output=True, text=True, timeout=10)
m = re.search(r'x\((.+)\);?\s*$', r.stdout, re.DOTALL)
if m:
    text = m.group(1)
    # 大整数字段除100转浮点
    for field in ["f43","f44","f45","f46","f47","f48","f57"]:
        text = re.sub(f'"{field}":(\d+)', lambda mv: f'"{field}":{float(mv.group(1))}', text)
    data = json.loads(text)
    items = data.get("data", {}).get("diff", [])
    zt = [i for i in items if i.get("f3", 0) >= 9.5]
    # f3=涨幅%，f8=换手率%，f10=振幅%
```

## 四、OpenClaw 核心命令

### 状态与诊断
```bash
openclaw status                     # 整体状态
openclaw status --all             # 详细状态
openclaw doctor                    # 健康诊断
openclaw health --verbose          # 详细健康检查
```

### Agent 管理
```bash
openclaw agents list --bindings   # 列出所有 agent 及绑定
openclaw agents list              # 列出所有 agent
```

### Cron 任务
```bash
openclaw cron list                # 列出定时任务
openclaw cron list --all         # 含禁用任务
openclaw cron run <id>           # 立即运行
openclaw cron runs               # 运行历史
```

### Gateway
```bash
openclaw gateway status           # 网关状态
openclaw gateway restart          # 重启网关
```

## 五、每日股扫标准流程

```
开盘前(08:30)  → morning_briefing.py → 飞书早报
开盘(09:00)    → market_open_scan.py → 跳空/异动检查
盘中每小时     → stop_loss_monitor.py → 止损检查
收盘(16:00)    → market_close_scan.py → 收盘报告 → 飞书
22:00          → daily-growth-review → 持仓分析+选股复盘
```
