# 06 - 股票业务规则

## MCP 工具接口

### 接口清单（82.156.17.205 MCP 服务器）

| 工具 | 功能说明 | 稳定性 |
|------|----------|--------|
| `brief` | 基本行情（代码/名称/PE/PB/ROE/价格/均价/振幅） | ✅ 主力 |
| `medium` | 中等详细（+行业概念/主力净流入） | ✅ 主力 |
| `full` | 完整数据（+30日技术指标历史） | ✅ 主力 |
| `industry_hot` | 板块热点 | ⚠️ session级（不稳定，重置后消失） |
| `LimitUp` | 涨停池 | ⚠️ session级（不稳定，重置后消失） |

### 调用方式

```python
import urllib.request, json

MCP_URL = "http://82.156.17.205/cnstock/mcp"

def mcp_call(tool, args, timeout=8):
    payload = {"jsonrpc":"2.0","method":"tools/call",
               "params":{"name":tool,"arguments":args},"id":1}
    req = urllib.request.Request(MCP_URL, data=json.dumps(payload).encode(),
        headers={"Content-Type":"application/json","Accept":"application/json, text/event-stream"})
    with urllib.request.urlopen(req, timeout=timeout) as resp:
        raw = resp.read().decode()
    idx = raw.find("data:")
    if idx < 0: return None
    result = json.loads(raw[idx+5:].strip()).get("result",{})
    return result if result else None
```

### 响应格式

```json
{
  "result": {
    "content": [{"type": "text", "text": "..."}]
  }
}
```

⚠️ **注意**: subprocess curl 对此 MCP 服务器返回 HTTP 501，必须用 urllib 直连。

## 业务规则

### 止损止盈
- 止损线：持仓成本 -8%（触发后次日开盘价卖出）
- 止盈线：持仓成本 +20%

### 选股标准
- 换手率 ≥ 3%
- RSI6 在 30-75 之间
- KDJ 超卖阈值：K < 20
- KDJ 超买阈值：K > 80
- MACD 金叉（3日内）

### MCP 数据字段
- main_net_inflow：主力净流入（**单位：亿元**，不是万元）
- f3：当日涨幅（不是昨日涨幅）
