## 问题描述

cron 定时任务触发时，main 独自执行，未调用 stock-exec/stock-review 等子 agent。文档里的协调链没有实际落地。

## 解决方案

1. 开盘扫描 cron 触发 → main 接收
2. main 通过 sessions_send 分派任务给 stock-exec
3. stock-exec 执行完成后，stock-review 验收
4. main 汇总结果，发送通知
