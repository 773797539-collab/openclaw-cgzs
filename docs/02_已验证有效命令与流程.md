## 7. MCP 股票查询（重要规范）

```python
# ✅ 正确方式：必须先 normalize_symbol
from mcp_utils import mcp_brief, normalize_symbol
text = mcp_brief(normalize_symbol('605365'))  # 返回完整数据

# ❌ 错误方式：直接传代码会返回 "No data found"
text = mcp_brief('605365')  # 返回 EMPTY
```

**原因**：MCP 服务器要求标准格式 `SHxxxxxx` 或 `SZxxxxxx`

**验证结果**：✅ normalize_symbol('605365') → 'SH605365'，mcp_brief 返回 1180 bytes

**已知约束**：
- `mcp_full()` 对部分股票（如 SH605365）返回空 → 用 `mcp_full_safe()` 容错降级
- MCP 健康检查：`mcp_health_check()` 返回 `(True, 27ms, None)`
