## 一、门户定位

门户是主可观测中心，关键状态、当前任务、通知摘要、异常、备份健康都集中在这里。

---

## 三、门户数据结构

```
portal/
├── index.md              # 总览页
├── status/              # 状态数据
│   ├── system.json      # 系统状态
│   ├── tasks.json       # 任务池快照
│   ├── backup.json      # 备份状态
│   └──异常.json         # 最近异常
└── templates/           # 门户模板
```
