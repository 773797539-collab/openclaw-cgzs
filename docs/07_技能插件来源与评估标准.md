## 评估标准

### 必备条件
1. **有 SKILL.md**：必须有标准化的 SKILL.md 文档
2. **用途明确**：触发条件清晰（description 里有触发词）
3. **可执行**：不是纯理论，有实际可操作的流程

### 优选条件
4. **有示例**：有参考代码/示例输出
5. **维护活跃**：近期有更新
6. **许可清晰**：非 Proprietary 或许可宽松

## 评估流程

1. 读取 SKILL.md，检查 description 和触发词
2. 评估是否适用于当前任务
3. 测试简单调用（不影响生产数据）
4. 记录到本表格
5. 高价值技能：添加到 TOOLS.md 参考

---

## clawhub.ai 安装方法

```bash
# 搜索技能
npx skills find <关键词>

# 安装技能（需要 GitHub package 路径）
npx skills add <owner/repo>

# 检查已安装技能更新
npx skills check
```
