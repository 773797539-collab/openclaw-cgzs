## 一、交易时间

| 时段 | 时间 |
|------|------|
| 上午 | 09:30 - 11:30 |
| 下午 | 13:00 - 15:00 |
| 集合竞价 | 09:15 - 09:25（开盘前）|
| 盘后定价 | 15:05 - 15:30 |

**非交易时间门控**：所有涉及实时数据的脚本，必须在开头加交易日判断：
```python
from datetime import datetime
def is_trading_day():
    return datetime.now().weekday() < 5  # 周一到周五

def can_run():
    """是否应该执行（交易日+交易时段）"""
    from datetime import datetime, time
    now = datetime.now()
    if now.weekday() >= 5:
        return False
    t = now.time()
    return (time(9, 25) <= t <= time(11, 30)) or (time(13, 0) <= t <= time(15, 0))
```

## 三、止损止盈配置

**文件**: `stock-assistant/data/alert_config.json`

| 字段 | 含义 |
|------|------|
| stop_loss | 负数，如 -0.15 表示亏15%止损 |
| take_profit | 正数，如 0.10 表示赚10%止盈 |

**止损计算**: 止损价 = 成本价 × (1 + stop_loss)
**实时止损检查**: `stop_loss_monitor.py` cron，每30分钟检查一次（周一至周五）

## 附录：涨跌停详细规则

### 涨跌停幅度

| 板块 | 涨跌幅限制 | 备注 |
|------|-----------|------|
| 主板（沪市/深市）| ±10% | 普通A股 |
| 创业板 | ±20% | 创业板注册制 |
| 科创板 | ±20% | 科创板注册制 |
| ST股 | ±5% | 特别处理股票 |
| *ST股 | ±5% | 退市风险警示 |
| 新股上市首日 | 无限制 | 但有盘中临停 |
| 北交所 | ±30% | 北交所注册制 |

### 涨停板识别

```python
def is_limit_up(price: float, change_pct: float) -> bool:
    """判断是否涨停"""
    if abs(change_pct) >= 9.9:  # 考虑小数精度
        return True
    # 创业板/科创板
    if change_pct >= 19.9:
        return True
    return False
```

### 涨停板实战要点

- 涨停板封板时间越早，次日溢价概率越高
- 首次开板（炸板）是卖点，不是买点
- 一字板（开盘即涨停）次日继续涨停概率大
- 连续涨停后第三个板开始关注特停风险（上交所10板/深交所7板）

## 技术指标选股标准

### 评分模型（tech_screen.py）

| 评分因素 | 信号 | 加分 |
|----------|------|------|
| KDJ 金叉（K<20超卖区） | 强势买入 | +30 |

| 评分因素 | 信号 | 减分 |
|----------|------|------|
| KDJ 死叉 | 趋势转跌 | -20 |
| MA 均线空头排列 | 下降趋势 | -15 |
| MACD 死叉 | 趋势转跌 | -10 |

### 推荐等级
- **buy**: score ≥ 80
- **watch**: 65 ≤ score < 80
- **neutral**: 30 < score < 65
- **sell**: score ≤ 30

### BOLL 止损公式
```
止损位 = min(BOLL下轨, 成本 × (1 - 最大亏损%), 买入价 - 2×ATR)
推荐止损 = 三者最小值
```

| 止损方式 | 公式 | 适用场景 |
|----------|------|----------|
| ATR止损 | 买入价 - 2×ATR | 波动大的股票 |
| BOLL止损 | BOLL Lower | 趋势交易 |
| 成本止损 | 成本 × 0.85 | 保守止损 |
