# 007 - 每周常设任务清单

**创建**: 2026-03-28
**目的**: 明确周一至周日各自该做什么，避免"无事可做"型停摆

---

## 周一（交易日）
- [ ] 00:30 止损监控触发（第一道防线）
- [ ] 08:30 morning_briefing（开盘前简报）
- [ ] 09:00 market_open_scan（开盘扫描）
- [ ] 全天 每30分钟 stop_loss_monitor 检查
- [ ] 16:00 market_close_scan（收盘报告）
- [ ] 收盘后：更新候选股池（stock_pool_updater）
- [ ] 收盘后：复盘今日操作，更新持仓历史

---

## 周二至周四（交易日）
- [ ] 每30分钟 stop_loss_monitor 检查
- [ ] 每日 market_close_scan
- [ ] 候选股池每日更新（stock_pool_updater）
- [ ] 持仓历史每日追加

---

## 周五（交易日）
- [ ] 所有日常任务
- [ ] 周度持仓复盘（portfolio_review）
- [ ] 候选股池周末清理
- [ ] 下周操作计划初步构思

---

## 周六/周日（休市）
- [ ] 每周常设任务清理（文档/代码/配置）
- [ ] MCP 工具能力摸底（工具研究）
- [ ] 候选股池数据质量检查
- [ ] 系统健康检查（脚本/磁盘/cron）
- [ ] 周末市场研究（板块/基本面）
- [ ] 下周开盘准备清单（周日下午完成）
- [ ] skills/plugins 研究（clawhub.ai）
- [ ] 复盘上周失败案例

---

## 每日常设（无论是否交易）
- [ ] 每15分钟 HEARTBEAT 自检
- [ ] 每次 commit 后：git push（自动或手动）
- [ ] 每日 backup cron（03:00）
- [ ] 每周 doc export（周日 03:00）

---

## 触发条件（需要人工介入时）
- stop_loss 触发 → 立即飞书通知
- MCP 全部失效 → 降级到 akshare
- Token 降至 20% → 提醒用户关注
- 连续3次 cron 失败 → 排查原因
